# Context

## 当前工作重点

项目核心功能和布局样式已稳定。

## 最近的更改

- **优化诗歌布局与对齐**：
  - 为诗歌容器设置了固定的最小高度，并使用 Flexbox 实现垂直居中，解决了切换诗歌时的布局跳动问题。
  - 通过 CSS 伪元素微调了作者署名的对齐，以达到更好的视觉居中效果。
- **重构情诗功能**：根据用户的多轮反馈，对情诗展示功能进行了全面的重构和优化。
- **更新诗歌库**：替换了全新的诗歌列表。
- **调整样式**：
  - 将字体更换为系统内置的楷体，并优化了字体备选列表。
  - 精确调整了诗歌正文和作者的字号、间距及透明度。
- **优化播放逻辑**：
  - 将情诗切换时间调整为10秒。
  - 实现了加载后立即随机显示一首诗的逻辑。
- **修复显示问题**：通过优化 CSS 策略，解决了背景视频偶尔不显示的白屏问题。
- **新增点击切换功能**：新增情诗区域点击切换功能，并优化了鼠标悬停样式。
